2全球宏观机遇：能源危机与算力缺口

2.1 海外能源短缺：AI发展的物理瓶颈

马斯克在2026年1月达沃斯论坛上明确指出："AI部署的根本限制因素是电力，就是能源。"AI瓶颈已从芯片转向电力，这一判断正在被数据不断验证。

海外：能源短缺制约算力扩张

  • 美国电力缺口：2028年数据中心电力需求74GW，可用仅25GW，缺口达49GW
  • 电网并网等待：美国数据中心电网接入审批周期3-4年
  • GPU交付周期：36-52周，CoWoS封装产能预订至2027年中
  • HBM紧缺：高带宽内存缺口延续至2028年
  • 欧洲电价高企：工业电价是中国3倍以上
  • 数据中心用电增速：2025年全球同比+17%，AI专用数据中心+50%

中国：电力与算力双重优势

  • 发电量领先：2026年中国发电量将达美国近3倍
  • 绿电成本极低：西部绿电0.3元/度（$0.04/kWh）
  • 风光发电占比：2025年超20%，预计2030年数据中心绿电占比超80%
  • 算力规模：2024年725.3 EFLOPS → 2026年1460.3 EFLOPS（翻倍）
  • 东数西算：已建成多个国家级算力枢纽节点
  • 算电协同：2023年写入政府工作报告，和林格尔示范项目新能源利用率92%

2.2 算力供需缺口量化

高盛预计，美国数据中心电力需求将从2025年的31GW增长至2027年的66GW，届时将吞噬美国夏季高峰电力需求的8.5%。国际能源署（IEA）数据显示，全球数据中心电力消耗在2026年将突破1000TWh。

2.3 成本优势的核心来源

中国算力的成本优势并非来源于芯片（实际上进口芯片受到限制），而是来源于电力成本和规模效应：

3Token出海市场：定义、规模与趋势

3.1 什么是Token出海

定义：Token出海是以Token（大模型推理的最小计量单位）为载体，将中国的算力、模型能力和智能服务体系打包为API服务，向全球客户跨境输出。本质是MaaS（模型即服务）模式的跨境交付，而非传统意义上的IDC出租或裸金属算力租赁。

与传统的算力出口模式相比，Token出海具有独特优势：

对比维度 传统IDC/算力出租 Token出海（MaaS）
交付物 裸金属服务器/虚拟机 API调用服务（按Token计费）
数据接触 客户数据进入我方基础设施 推理结果输出，数据可不出境
出口管制风险 高（涉及芯片/硬件出口） 低（数字服务，受WTO电子传输关税豁免）
客户门槛 高（需懂运维） 低（API调用即可）
毛利水平 20-40%（IaaS） 40-80%（MaaS）
规模效应 线性 指数级（推理优化摊薄成本）

4.3 竞争差异化分析

市场现有玩家的不足

  • 超大规模云（AWS/Azure/GCP）：价格高昂，受限于美国电力短缺难以扩容，排他性强
  • 新兴GPU云（CoreWeave/Lambda）：规模有限，依赖英伟达供货，受制于GPU供应链紧缺
  • 中国互联网云（阿里/腾讯/字节）：虽有成本优势，但以商业利润为导向，对政府/科研市场投入不足
  • 超算领域：海外商业超算服务稀缺，AWS HPC价格极高且非专用

深圳国企的差异化空间

  • "超算+智算"双引擎：唯一同时具备2000P FP64超算和20000P智算的国企，可覆盖科研仿真+AI推理双场景
  • 国企信用背书：在数据安全、合规信誉方面具有天然信任优势，适合政府/科研/军工等敏感客户
  • 鹏城实验室生态：与鹏城云脑、鹏城实验室深度联动，拥有国家级科研平台背书
  • 深圳产业集群：依托深圳AI全链生态，可提供"算力+模型+应用"一体化方案
  • 超算服务蓝海：FP64超算作为服务输出，全球几乎没有直接竞品
战略空白点：当前Token出海市场主要由互联网企业主导，聚焦通用AI推理服务。但高性能计算（HPC）作为服务的出海市场几乎空白——海外科研机构和企业对FP64超算算力有旺盛需求，但AWS/GCP的HPC服务价格极高且非专用。贵司2000P FP64超算资源可切入这一蓝海市场，与智算Token出海形成"双引擎"组合，构建独特的竞争壁垒。

5.3 "硬件+软件+信用"三位一体竞争力

单纯的算力出租容易被复制和价格战消耗，需构建多层次的竞争壁垒：

6.2 L1 - 智算Token API（核心规模层）

这是Token出海的主体业务，利用20000P智算资源为海外客户提供大模型推理服务。

  • 模型选择：以通义千问、DeepSeek等国产开源大模型为基座，这些模型在OpenRouter上已验证全球竞争力
  • 计费体系：采用分层定价（基础$3/进阶$6/高速$45/研究$150），对标国际但保持3-5倍成本优势
  • 技术核心：推理优化引擎（KV Cache优化、批处理、量化压缩）是利润率的关键——CoreWeave凭借优化技术实现60% EBITDA利润率
  • 交付方式：RESTful API + SDK，支持流式输出、Function Calling、多模态
  • 数据安全：客户输入数据仅用于推理，不做存储/训练，推理完成后即销毁

6.3 L2 - 超算HPCaaS（差异化利润层）

这是贵司最独特的竞争优势——2000P FP64超算算力在全球几乎没有直接竞品。

  • 应用场景：计算流体力学（CFD）、分子动力学模拟、气象/海洋预报、药物筛选、材料科学、核物理仿真、地质勘探
  • 目标客户：海外高校超算中心溢出需求、工业研发部门（航空/汽车/制药）、国家级科研项目
  • 定价参考：AWS HPC按需实例约$3-5/核心·小时，贵司可定价$1-2/核心·小时，保持2-3倍成本优势
  • 交付方式：Web Portal提交作业 + MPI/Slurm调度 + 结果可视化下载
  • 附加价值：可提供算力+算法+专家咨询的一体化服务，大幅提升客单价
超算出海的蓝海逻辑：全球高性能计算服务市场规模预计2034年将达到数百亿美元，但主要被欧美超算中心自用。商业化的HPCaaS服务几乎被AWS/GCP垄断且价格极高。贵司2000P FP64超算若以服务形式输出，将切入一个供给极度稀缺、价格敏感度低的蓝海市场。

6.4 L3 - 行业解决方案（高附加值层）

  • 多语言模型本地化：复制"新加坡海狮"模式，为东南亚、中东、非洲等地区提供本地化大模型训练服务
  • 政府级AI基础设施：为海外政府提供AI算力+模型+安全方案的整体交钥匙工程
  • 科研合作计划：与海外高校联合研究，提供算力支持换取学术影响力和数据合作
  • 行业垂直模型：面向石油勘探、气象服务、生物制药等行业的专用模型训练与部署

8技术架构与平台建设方案

8.1 整体技术架构

Token出海平台需要构建五大核心模块：

模块 功能 关键技术 建设周期
1. API网关层 请求路由、鉴权、限流、计费 Kong/APISIX + JWT + Rate Limiting 2个月
2. 推理服务层 模型加载、推理执行、结果返回 vLLM/TensorRT-LLM + 推理优化引擎 3个月
3. 超算调度层 HPC作业提交、Slurm调度、结果管理 Slurm + Web Portal + 可视化引擎 4个月
4. 计费与结算系统 多币种计费、跨境收款、发票管理 Stripe/Paddle集成 + 多币种清算 3个月
5. 安全与合规模块 内容审核、数据隔离、日志审计 内容过滤 + 数据脱敏 + 审计日志 持续

8.2 推理优化引擎（利润率核心）

推理优化技术直接决定Token出海的利润率。CoreWeave凭借优化技术实现60% EBITDA利润率，贵司需重点投入以下方向：

  • 批处理优化：Continuous Batching技术，动态批合并请求，提升GPU利用率至80%+
  • KV Cache优化：PagedAttention技术，减少显存碎片，提升并发能力2-3倍
  • 量化压缩：INT8/INT4量化，在精度损失可控的前提下降低推理成本50%+
  • 模型蒸馏：大模型→小模型蒸馏，为不同价格层级提供差异化模型
  • 国产芯片适配：针对昇腾NPU优化推理框架，充分发挥国产芯片性能

8.3 数据安全与隔离架构

核心原则：客户数据不存储、不训练、不出境。推理完成后立即销毁输入数据，仅保留计费日志（不含原始内容）。
  • 推理层隔离：每次推理请求使用独立沙箱，推理完成后内存清零
  • 日志脱敏：计费日志仅记录Token数量、时间戳、API Key，不记录请求/响应内容
  • 网络隔离：推理集群与互联网之间部署API网关，不开放SSH/直接访问
  • 合规审计：全链路操作日志，支持第三方安全审计
  • 数据主权声明：在服务条款中明确"数据处理在中国境内完成，客户数据不被存储或用于训练"

8.4 边缘节点部署

为降低海外用户访问延迟，需要在目标市场部署轻量级边缘节点：

  • 新加坡节点：覆盖东南亚，API响应延迟<50ms
  • 迪拜节点：覆盖中东，API响应延迟<80ms
  • 东京/法兰克福节点：覆盖日韩/欧洲（后期）
  • 架构：边缘节点仅做API路由和缓存，实际推理在深圳中心节点完成
  • 加速方案：利用中国-东南亚海缆系统（如SMW5/SEA-ME-WEA6）保障网络质量

9政策合规与风险防控

9.1 合规框架总览

Token出海涉及中国和目标市场双重合规要求，需要系统设计合规体系：

合规领域 中国法规 要求与应对
数据出境 《数据安全法》《个人信息保护法》 Token出海模式下，客户数据不进入中国境内（推理后即销毁），规避数据出境评估。若涉及用户数据，走标准合同或安全评估路径
AI服务管理 《生成式AI服务管理暂行办法》 该办法主要针对境内向公众提供的服务，出海API服务需在内容审核方面做好合规（对海外客户的请求进行内容安全过滤）
出口管制 《出口管制法》及相关清单 Token API服务属于数字服务，不涉及芯片/硬件出口。但需确认目标客户不在制裁名单上
国企监管 国资委相关规定 需完成国企境外投资/业务审批，可通过设立市场化子公司或参股方式运营海外业务

9.2 "模型出海"间接实现"算力出海"

关键策略：美国BIS（工业与安全局）已对华为昇腾芯片发布全球禁令，直接出口芯片或裸金属算力将面临制裁风险。但通过"模型出海"间接实现"算力出海"——即向海外客户提供模型API服务而非算力租赁——可有效规避芯片管制。客户获得的是API调用能力，而非对中国芯片的直接使用。

9.3 海外实体架构设计

建议通过多层架构实现合规运营：

  • 第一层（境内主体）：深圳国企母公司持有算力资产和核心知识产权
  • 第二层（香港中间层）：设立香港全资子公司，作为国际业务总部，负责资金结算和合同签署
  • 第三层（目标市场实体）：在新加坡/迪拜等地设立本地运营公司，负责本地客户服务和合规对接
  • 资金流：海外客户 → 本地运营公司 → 香港子公司 → 境内主体（技术服务费）
  • 数据流：海外客户 → API请求 → 深圳推理集群 → 结果返回（数据不落地存储）

9.4 国际法规应对

法规 影响 应对
美国BIS出口管制 昇腾芯片全球禁令 不出口芯片/硬件，仅提供API服务；筛查客户是否在实体清单
欧盟AI法案 对高风险AI系统有透明度要求 提供模型说明文档，标注训练数据来源；基础推理服务属低风险
GDPR（欧盟） 个人数据处理要求 推理后数据即销毁，不存储个人信息；签署SCC标准合同条款
WTO电子传输免关税 数字服务跨境传输关税豁免 Token API服务属于电子传输，享受免关税待遇

9.5 内容安全与审核

向海外客户提供API服务时，需在内容安全方面做好平衡：

  • 输入过滤：对客户请求进行内容安全审核，过滤违法有害内容（依据中国法规和目标市场法规双重标准）
  • 输出审核：对模型生成结果进行安全过滤，避免输出违规内容
  • 分级策略：不同市场采用不同审核标准，在合规与用户体验之间平衡
  • 审计留痕：所有审核决策可追溯，便于应对监管审查

风险防控总体策略

风险防控三原则：
  1. 不把鸡蛋放在一个篮子里：市场多元化（东南亚+中东+日韩），业务多元化（Token+超算+方案），客户类型多元化
  2. 从设计源头降风险：数据不落地存储、不出口芯片硬件、通过模型API间接输出算力——从业务模式设计上就规避最大风险
  3. 快速试错、小步快跑：先以最小可行产品（API服务）验证市场，再逐步投入超算和行业方案，避免一次性大规模投入

关键成功因素

  1. 推理优化技术能力——这是利润率的核心，决定能否在价格战中存活
  2. 超算服务差异化——这是唯一不被互联网大厂复制的竞争优势
  3. 海外合规能力——合规是出海的生命线，国企身份既是优势也是敏感点
  4. 快速市场验证——6个月内上线API服务，用真实用户反馈驱动产品迭代
  5. 生态合作能力——与鹏城实验室、高校、行业协会、目标市场本地企业建立合作网络
